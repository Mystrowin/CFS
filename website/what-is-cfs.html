What is CFS? A compressed archive that opens like a Windows folder.

CFS is experimental Windows software for working with a single .cfs compressed archive as a folder-like workspace in File Explorer. It uses Windows Projected File System for on-demand access and saves supported changes back into the archive.

01

How the workflow operates

Open, use, save, and unmount.

01

Open one archive

CFS reads the selected .cfs archive and prepares a managed workspace for it.

02

Browse in Explorer

ProjFS presents the archive contents as files and folders. File data is supplied when Windows or an application requests it.

03

Save supported changes

CFS validates and writes supported workspace changes back into the same archive before closing the mount.

02

Why the project exists

A folder-style workflow without abandoning compression.

A

One user-managed file

The long-term product promise is that the user manages one compressed archive rather than a permanent extracted folder beside it.

B

On-demand file access

Explorer mounting avoids requiring a complete up-front extraction when the ProjFS workflow is available.

C

Persistent workspace edits

Supported file operations are preserved in the archive when the workspace is saved and reopened.

System requirements

NOT A BACKUP REPLACEMENT

Know the beta boundary before testing.

CFS is not production-ready and does not promise compatibility with every Windows application. Mounting, saving, performance, and data-integrity bugs may still exist.

03

Common questions

What testers should know first.

Does CFS extract everything first?

Not during ProjFS mounting. CFS projects the directory structure and supplies file contents on demand. A full-extraction compatibility mode is available when necessary.

Can files be edited?

CFS provides a folder-style editing workflow. Supported changes are written back when the workspace is saved and unmounted.

Is the beta free?

The current installer and portable packages are available at no charge from the public GitHub release page.
